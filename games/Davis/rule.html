Rules and How to Play

1. Add a random number IN EACH box between 0-9
of your choice to the 3 grey box

2.1 If a red box is shown then that mean you guess to high

2.2 If a blue box is shown then that mean you guess to low

2.3 If the box with the question mark turn green and
show a number you got it right

3.Whatever number is in this box is the number of guess left

4. Final rule have fun
